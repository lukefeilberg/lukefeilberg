Projects

The Onion Classifier

Using the Pushshift API for Reddit a dataset is created consiting of 9,000 Onion headlines and 15,000 "Onion-like" headlines from the subreddit r/NotTheOnion. Then Tensorflow/Keras is used to build some different neural networks to try to classify the headlines as real Onion or not, achieving 87% validation accuracy.


Boggle

Play Boggle on any size board here! The board is generated using Javascript and the longest words are found in an efficient manner using a Trie data structure which is explained in more detail beneath the Boggle board.


Voter Turnout/Suppression Regression

Voter ID laws have caused concern claiming they may suppress voter turnout, particularly in counties that have DMV's open only one day a week or inconveniently located far away. Data is downloaded and scraped from the web to investigate whether the total hours DMV's are open and their average distance to poorer neighborhoods influences voter turnout in Alabama, a state with strict voter ID laws.


Quick COVID-19 Map

Using GeoPandas, Johns Hopkins COVID-19 data and US Census data I decided to look at how the cases of COVID-19 per milllion people evolved in the US over time.


Twitter Scraper

As someone who eats a plant-based diet and interested in plant-based meat alternatives I wanted to compare people's opinions between the Impossible Burger and the Beyond Burger. To do this I use Selenium to scrape tweets containing either the phrase "beyond burger" or "impossible burger" then use VADER Sentiment Analysis to estimate the average sentiment of each tweet, which I finally compare through some regressions and visualizations.


Blackjack Reinforcement Learning

Using reinforcement learning to try to learn the optimal blackjack playing strategy. In particular, the game is modeled as a Markov Decision Process and then Policy Iteration is performed to learn under what circumstances should someone draw or stop.


Funky Party Robot

A silly project making a robot "sing" with a piezo crystal and "dance" with a servo motor and LED lights.
